Definition: Linear Function

A linear function is a polynomial of degree \(1\) or \(0\), i.e. a function \(f:\mathbb R\to\mathbb R\) of the form \[f(x):=a_1x+a_0,\quad\quad a_0,a_1\in\mathbb R.\]
