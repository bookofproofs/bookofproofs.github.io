Definition: Language

Let $(\Sigma^*,\cdot)$ be the set of all strings over an alphabet $\Sigma$ with the concatenation operation "$\cdot$". Any non-empty subset \(L\subseteq (\Sigma^*,\cdot)\) is called a language over the alphabet \(\Sigma \).
