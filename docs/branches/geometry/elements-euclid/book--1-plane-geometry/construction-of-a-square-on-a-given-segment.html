Proposition: 1.46: Construction of a Square on a Given Segment

(Proposition 46 from Book 1 of Euclid's “Elements”)

To describe a square on a given straight line.

  • Let $AB$ be the given straight line.
  • So it is required to describe a square on the straight line $AB$.

Modern Formulation

Given an arbitrary segment \(\overline{AB}\), it is possible to construct a square on that segment.
