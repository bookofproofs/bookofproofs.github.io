Definition: Surjective Function

A function \(f:A\mapsto B\) is called surjective (or surjection), if for every \(y\in B\) there is an \(x\in A\) with \(f(x)=b\). This corresponds to the right-total property, in addition to its the properties of a function.

Notes

  • A surjective function aims at every element $b\in B.$
  • In other words, every element $b\in B$ has a fiber.
  • In other words, the image $f(A)$ is the whole set $B.$
