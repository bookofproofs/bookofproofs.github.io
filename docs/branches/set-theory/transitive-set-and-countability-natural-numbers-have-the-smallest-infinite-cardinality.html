Explanation: Transitive Set and Countability - Natural Numbers Have the Smallest Infinite Cardinality

(related to Part: Cardinal Numbers)

The above examples as well as the definition of infinite sets show that natural numbers $\mathbb N$ play a prominent role in the study of the cardinality of infinite sets. This is not a coincidence. In the next branch of BookofProofs, the number systems and arithmetics, we will provide a set-theoretic definition of natural numbers, in which we will be using the inductive set $\omega$ we have learned about when talking about the Zermelo-Fraenkel axioms of set theory. Thus, we can recap the following facts:

  • The minimal inductive set $\omega$ is used to define the natural numbers $\mathbb N$. By this definition, they have the same cardinalities: $|\mathbb N|=|\omega|.$
  • When comparing cardinalities, we have learned that for two given cardinalities the relation $|A|\le |B|$ is fulfilled if there is an injective function $f:A\to B.$
  • We have shown that $\omega$ is minimal in the sense that it is the subset of any existing inductive set.
  • There is always an injective function $f:A\to B$, if $A\subseteq B.$ For instance, take the identity function $f(a)=a$ for all $a\in A.$

These facts show us that the natural numbers $\mathbb N$ have the smallest cardinality of all infinite sets. The big question now is:

bq{color:blue}. Is this the only possible cardinality of infinite sets or are there infinite sets with bigger cardinalities?

In order to make some progress in answering this question, we have to go back again to the definition of comparing cardinal numbers.
