Definition: Power Set

Let $X$ be a set. The axiom of power set and the axiom of empty ensure * the existence of the power set, \(\mathcal P(X) :=\{A\mid A\subseteq X\}\), i.e. the set of all possible subsets of \(X\)1, and
* that the empty set \(\emptyset\) is a subset any non-empty set \(X\).
