BookofProofs

About

BoP is an open book dedicated to mathematics, physics, and computer science. Its goal is to broaden the public knowledge of the axiomatic method.

Motto

The ability to formulate mathematical proofs using the axiomatic method should be taught as a basic skill like reading or writing.

This is what BoP is all about!

Project

BoP was launched in February 2014 is a collaborative project. In February 2023, we migrated the site to github. You can now co-author if you are a github user.

Open-source and ad-free content

The main differences to related mathematical resources available on the internet are that BoP is guaranteed ad-free, everybody can contribute on GitHub ;-), and you can use some built-in features like Sagecell and JSXGraph for interaction with complex mathematical concepts.
