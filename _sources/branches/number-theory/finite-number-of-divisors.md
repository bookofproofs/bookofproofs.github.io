If `\(a \neq 0\)` is an [integer][bookofproofs$844] and `$b$` its [divisor][bookofproofs$700] `\(b\mid a\)`, then `\(|b|\le|a|\)`. In particular, each `\(a\neq 0\)` has only a [finite][bookofproofs$985] number of [divisors][bookofproofs$1273].
