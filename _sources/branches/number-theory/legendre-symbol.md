Let `$p > 2$` be an [odd][bookofproofs$8130] and fixed [prime number][bookofproofs$473]. The **Legendre symbol** `$p$` `$\left(\frac np\right)$` is an [arithmetic function][bookofproofs$8112] defined using the [quadratic residues modulo][bookofproofs$8196] `$p$` as follows:
`$$\left(\frac np\right):=\begin{cases}
1&\text{if }n\text{ is quadratic residue modulo }p\text{ and }p\not\mid n,\\
-1&\text{if }n\text{ is a quadratic nonresidue modulo }p\text{ and }p\not\mid n,\\
0&\text{if }p\mid n.
\end{cases}$$`

### Examples

* If `$n$` is odd and `$p\not\mid m$`, then `$\left(\frac {n^2}p\right)=1.$`
* `$\left(\frac 1p\right)=1$` for all `$p > 2.$`
