The [derivative][bookofproofs$1370] of the [exponential function][bookofproofs$281] `$f(x)=\exp(x),$` `$x\in\mathbb R,$` is `$$f'(x)=\exp(x).$$`

The exponential function is the only function, which equals its own derivative.
