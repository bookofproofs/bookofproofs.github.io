The [absolute value function][bookofproofs$6681] `$f(x)=|x|,$` `$x\in\mathbb R,$` has no [derivative][bookofproofs$1370] at the point `$x=0.$` However, it has a [left- and a right-derivative][bookofproofs$6762] at this point.
