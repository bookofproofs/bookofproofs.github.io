The **exponential series**

`\[\sum_{n=0}^\infty\frac{x^n}{n!}\]`

is an [absolutely convergent series][bookofproofs$198]  for every [real number][bookofproofs$1105] `\(x\in\mathbb R\)`. It defines a [function][bookofproofs$592] `\(\exp:\mathbb R\mapsto \mathbb R\)`, called the **exponential function** for all `\(x\in\mathbb R\)`.

`\[\exp(x):=\sum_{n=0}^\infty\frac{x^n}{n!},\quad\quad x\in\mathbb R.\]`
