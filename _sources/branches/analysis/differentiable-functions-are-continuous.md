Let `\(D\subseteq\mathbb R\)` (`\(D\)` is a [subset][bookofproofs$552] of [real numbers][bookofproofs$1105]). If a function `\(f:D\to \mathbb R\)` is [differentiable][bookofproofs$1370] in `\(D\)`, then `\(f\)` is [continuous][bookofproofs$219] in `\(D\)`.
