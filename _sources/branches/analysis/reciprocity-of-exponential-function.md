The [exponential function][bookofproofs$281] is never zero (non-zero property) `\[\exp(x)\neq 0\]` for all `\(x\in\mathbb R\)`, and it fulfills the following reciprocity law:

`\[\exp(-x)=\frac1{\exp(x)}.\]`
