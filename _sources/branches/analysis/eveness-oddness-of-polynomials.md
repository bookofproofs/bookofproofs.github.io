A [polynomial][bookofproofs$199] `$p:\mathbb R\to\mathbb R,$`
`$$p(x)=a_nx^n + \ldots + a_1x + a_0,\quad(a_k\in\mathbb R),$$` is [even (respecively odd) ][bookofproofs$416], if and only if for all [odd][bookofproofs$2318] (respectively [even][bookofproofs$2317]) indizes `$k$` we have `$a_k=0.$`
