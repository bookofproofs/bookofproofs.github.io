A set `\(M\)` with a [`\(\sigma \)`-algebra][bookofproofs$6212] `\({\mathcal {A}}\)` defined on it as well as all of its [subsets][bookofproofs$552] `\(T\subseteq M\)` are called **measurable sets**.
