The exponential function of [general base][bookofproofs$1603] `\(a > 0 \)` is never zero (non-zero property) `\[\exp_a(x)\neq 0\]` for all `\(x\in\mathbb R\)`, and it fulfills the following reciprocity law:

`\[\exp_a(-x)=\frac1{\exp_a(x)}.\]`
