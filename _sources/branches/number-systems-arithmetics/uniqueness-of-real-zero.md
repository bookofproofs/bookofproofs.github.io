Among the [real numbers][bookofproofs$1105] `$\mathbb R$`, there is only one [number zero][bookofproofs$34] `\(0\)`, which is neutral with respect to [addition][bookofproofs$1514], i.e. for which `$x=x+0=0+x$` holds for all `$x\in\mathbb R$`.
