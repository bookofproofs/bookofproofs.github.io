An [family of sets][bookofproofs$6198] `\(X_{i},i\in I,\)` is called *mutually [disjoint][bookofproofs$7985]* (or *pairwise [disjoint][bookofproofs$7985]*), if for all `\(i,j\in I\)`, `\(i\neq j\)`, `\(X_{i}\cap X_{j}=\emptyset .\)`
