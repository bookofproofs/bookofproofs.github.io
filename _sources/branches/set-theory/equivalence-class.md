The elements `$x$` of the [set][bookofproofs$550] `$V$`, which are [equivalent][bookofproofs$574] to a given element `$a\in V$`, i.e. the set `$[a]:=\{x\in V, x\sim a\}$` is called the **equivalence class** of `\(a\)`.
