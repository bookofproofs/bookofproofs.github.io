Let `$A$` and `$B$` be two [sets][bookofproofs$550]. Then the [intersection][bookofproofs$6828] `$A\cap B$` is a [subset][bookofproofs$552] of both, `$A$` and `$B$`, formally `$A\cap B\subseteq A$` and  `$A\cap B\subseteq B$`.
