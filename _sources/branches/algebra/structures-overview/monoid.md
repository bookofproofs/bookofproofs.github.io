A **monoid** is a [semigroup][bookofproofs$660] `\((X,\ast)\)` with an [identity][bookofproofs$661], i.e. an element `$e\in X$` with `$x=e\ast x=x\ast e$` for all `$x\in X.$`

Please note the [uniqueness of the neutral element][bookofproofs$669] `$e\in X$`.
