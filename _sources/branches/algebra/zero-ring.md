A [ring][bookofproofs$683] `\((R, +,\cdot)\)` is called a **zero ring** (or **trivial ring**), if 
`\[(R, +,\cdot)=(\{0\}, +,\cdot),\]`
i.e. if it contains only the element `\(0\)`.
