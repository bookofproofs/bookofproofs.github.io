Let `\(G(V,E,\gamma)\)` be an [undirected graph][bookofproofs$523].
* `\(G\)` is called a **forest**, if it contains only [acyclic][bookofproofs$6376] [components][bookofproofs$1221].
*  `\(G\)` is called a **tree**, if it has only one acyclic component.
